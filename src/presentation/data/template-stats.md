# 模板库统计

## 模板概览

当前模板库包含 **13 个** 精心设计的模板，涵盖多个常用场景。

## 按类别分类

### 📄 页面模板 (3 个)

- **产品着陆页** - 现代化产品展示页面，包含英雄区、特性展示、客户评价和行动召唤
- **关于我们页面** - 企业介绍页面，包含团队介绍、公司历史和价值观展示
- **联系我们页面** - 联系页面模板，包含联系表单、公司信息和地图展示

### 📊 仪表盘模板 (2 个)

- **数据分析仪表盘** - 完整的数据分析仪表盘，包含 KPI 卡片、图表和数据表格
- **管理后台仪表盘** - 企业级管理后台，包含用户管理、系统设置和操作日志

### 📝 表单模板 (3 个)

- **用户登录表单** - 简洁的登录表单，包含用户名、密码和记住我选项
- **用户注册表单** - 完整的用户注册表单，包含个人信息验证和条款确认
- **联系表单** - 通用联系表单，包含姓名、邮箱、主题和消息字段

### 🛒 电商模板 (1 个)

- **产品展示页面** - 电商产品详情页面，包含产品图片、描述、价格和购买按钮

### 📝 博客模板 (1 个)

- **博客文章布局** - 标准博客文章页面，包含文章标题、内容、作者信息和相关文章推荐

### ❌ 错误页面 (1 个)

- **404 错误页面** - 友好的 404 错误页面，包含错误信息和导航链接

## 模板特点

### ✅ 优势

1. **实用性强** - 所有模板都是实际项目中常用的场景
2. **结构完整** - 每个模板都包含完整的组件层次结构
3. **样式现代** - 使用现代化的设计风格和配色方案
4. **响应式** - 考虑了不同屏幕尺寸的适配
5. **可定制** - 所有组件属性都可以根据需要调整

### 🎯 使用场景

- **企业官网** - 着陆页、关于我们、联系我们
- **管理系统** - 登录、注册、仪表盘、管理后台
- **电商平台** - 产品展示页面
- **内容网站** - 博客文章布局
- **错误处理** - 404 错误页面

## 技术实现

### 组件类型使用统计

- **container** - 容器组件，用于布局和分组
- **text** - 文本组件，用于标题、描述等
- **button** - 按钮组件，用于交互操作
- **input** - 输入组件，用于表单字段
- **card** - 卡片组件，用于内容展示

### 属性特点

- **响应式设计** - 使用百分比和 rem 单位
- **现代配色** - 使用 Tailwind CSS 色彩系统
- **层次结构** - 清晰的父子组件关系
- **可访问性** - 考虑了语义化和可访问性
