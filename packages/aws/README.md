# @lowcode-platform/aws

AWS 集成工具库，提供 S3、CloudFormation、Lambda、API Gateway、CloudFront 等 AWS 服务集成功能。

## 功能

- **S3 操作**: 上传、下载、删除、列表
- **CloudFormation**: 模板生成
- **Lambda**: 函数部署
- **API Gateway**: API 配置
- **CloudFront**: CDN 配置
- **IAM**: 权限管理辅助

## 安装

```bash
pnpm add @lowcode-platform/aws
```

**注意**: 需要安装 AWS SDK peer dependencies。
